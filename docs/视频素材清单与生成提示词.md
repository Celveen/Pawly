# 视频素材清单与 AI 生成提示词

## 放置规则（放进去即生效，无需改代码）

| 位置 | 文件路径 | 建议规格 |
|---|---|---|
| 首页首屏全屏背景 | `public/videos/hero.mp4` | 16:9 · 1920×1080 · 8~12 秒 · 无声 |
| 首页会员横幅底衬 | `public/videos/member.mp4` | 16:9 · 1280×720 · 6~10 秒 · 无声 |
| 首页科普章节小窗 | `public/videos/journal.mp4` | 16:9 · 1280×720 · 6~10 秒 · 无声 |

- 格式：**MP4（H.264 编码）**，网页兼容性最好；单个文件尽量压到 **8MB 以内**（首屏的可放宽到 15MB），可用 [HandBrake] 或 ffmpeg 压缩。
- 视频会**静音自动循环播放**，所以生成时不用管声音；**首尾画面越接近，循环越无缝**（生成时在提示词里写"循环镜头/seamless loop"，或选平缓运镜）。
- 没放视频时页面显示流动光斑占位，不影响上线。

## 风格基准（三段视频共用的前缀）

> 温暖自然的生活方式短片，奶油色与浅绿色调，柔和晨光，浅景深，画面干净无杂物，缓慢平稳的镜头运动，真实质感，无文字无水印无 logo，电影感调色

负面提示：文字、字幕、水印、logo、快速剪辑、镜头抖动、过饱和、人脸特写

## 各视频具体提示词

### 1. hero.mp4（首屏全屏背景 —— 最重要，值得多生成几版挑）

> 【风格前缀】+ 阳光洒进的客厅或草地上，一只金毛犬与一只橘白猫安静地并排趴卧，微风轻轻吹动毛发，猫咪缓慢眨眼，狗狗耳朵偶尔轻动，背景虚化的绿植，镜头极缓慢地平移推进，10 秒无缝循环

要点：
- 画面**中部偏上要相对干净**（大标题压在上面），主体宠物最好位于画面下半部或两侧
- 色调必须偏奶油绿（和网站底色 #EFF3EB 融合），不要冷蓝调
- 多生成几版：也可以试「幼猫爪子轻拍慢食碗」「狗狗在草地上慢动作奔跑（远景）」

### 2. member.mp4（会员横幅底衬 —— 深色场景）

> 【风格前缀】+ 暖光夜晚的家中，主人手部轻抚熟睡猫咪的慢动作特写，深绿色调暗背景，光影柔和，6 秒无缝循环

要点：画面整体要**偏暗**（上面压深绿渐变+白字），亮部不要超过画面 1/3。

### 3. journal.mp4（科普章节小窗 —— 轻快学习感）

> 【风格前缀】+ 阳光书桌上，一只小猫好奇地用爪子碰翻开的书页，旁边放着眼镜和绿植，特写镜头，轻快温馨，6 秒无缝循环

## 注意事项

1. **平台选择**：即梦、可灵、Vidu 等均可，注意选择**允许商用**的会员档位并留存生成记录（工单/订单截图），上线后素材版权可追溯。
2. **循环处理**：如果生成的片段首尾跳变明显，可用剪辑软件把片段"正放+倒放"拼接成回旋镜头再导出（网站端也可以后续加回旋播放逻辑，先拼好最省事）。
3. **压缩命令参考**：`ffmpeg -i in.mp4 -vcodec libx264 -crf 26 -preset slow -an -movflags +faststart out.mp4`（`-an` 去音轨，`faststart` 让网页边下边播）。
4. 替换或新增视频后**无需改代码、无需重新构建**，刷新页面即生效（部署在服务器/Vercel 时需要随代码库提交或放到静态目录）。
